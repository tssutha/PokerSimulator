This is a python learning exercise. 

The Pokerhand.py contains code for Simple Poker game simulator.

[Read more about poker hand ranking here](http://en.wikipedia.org/wiki/Hand_rankings)


P  - One Pair
2P - Two Pair
3K - Three of a Kind
S  - Straight
F  - Flush
FH - Full House
4K - Four of a Kind
SF - Straight Flush
